Connecting Sendcloud lets your AI agents look up shipments and tracking data from Sendcloud.

## What you'll need

- **API key** (public key) and **API secret** (secret key) — a Sendcloud API key pair.

## Create your Sendcloud API key

<Steps>
  <Step title="Open the integrations settings">
    In the Sendcloud panel, go to **Settings \> Integrations** and choose the **Sendcloud API** integration.
  </Step>
  <Step title="Create a key pair">
    Create a new API key. Sendcloud generates a **Public key** and a **Secret key** — copy both.
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Sendcloud**, and fill in your API key (public key) and API secret (secret key).

<Warning>
  Keep your API secret safe and only paste it into Minimal AI's Sendcloud integration.
</Warning>
