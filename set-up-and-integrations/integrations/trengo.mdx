---

Connecting Trengo lets your AI agents read and reply to conversations in your Trengo team inbox.

## What you'll need

- **API token** — a personal access token from your Trengo settings.

## Create your Trengo API token

<Steps>
  <Step title="Open the REST API settings">
    In Trengo, click **Settings** in the lower-left corner, then go to **Integrations \> Rest API**.
  </Step>
  <Step title="Generate the token">
    Create (or regenerate) your personal access token and copy it.
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Trengo**, and paste your API token.

<Warning>
  Regenerating the token invalidates the old one. Keep it secret and only paste it into Minimal AI's Trengo integration.
</Warning>
