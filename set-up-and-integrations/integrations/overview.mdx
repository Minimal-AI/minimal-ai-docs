Pick the systems you use below. Each page lists the exact credentials you'll need and where to find them, then how to connect in Minimal AI.

## How connecting works

<Steps>
  <Step title="Gather your credentials">
    Open the integration's page below to see the exact fields you'll need (for example an API key, subdomain, or access token) and where to generate them in the provider.
  </Step>
  <Step title="Connect in Minimal AI">
    Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), pick the integration, and paste in the credentials.
  </Step>
  <Step title="Assign it to an agent">
    Once connected, enable the integration on the relevant AI Agent so it can use the data and actions in conversations.
  </Step>
</Steps>

<Info>
  Treat API keys and tokens like passwords. Create a dedicated key for Minimal AI where possible, grant the minimum permissions required, and never share credentials over email or chat.
</Info>
