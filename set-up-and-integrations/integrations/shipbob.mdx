Connecting ShipBob lets your AI agents look up fulfillment orders and shipment status from ShipBob.

## What you'll need

- **API key** — a ShipBob personal access token.

## Create your ShipBob API key

<Steps>
  <Step title="Open your integrations">
    Log in to the ShipBob merchant dashboard and go to **Integrations \> API tokens** (Personal Access Tokens).
  </Step>
  <Step title="Generate a token">
    Generate a new token, give it a name (for example, `Minimal AI`), and copy it.
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> ShipBob**, and paste your API key.

<Warning>
  Keep your token secret and only paste it into Minimal AI's ShipBob integration.
</Warning>
