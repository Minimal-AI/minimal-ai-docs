Connecting Dixa lets your AI agents read and reply to conversations in your Dixa workspace.

## What you'll need

- **API token** — a Dixa API token created in your Dixa settings.
- **API base URL** (optional) — leave the default unless Dixa told you to use a specific URL.

## Create your Dixa API token

<Info>
  Only Dixa **administrators** can generate API tokens.
</Info>

<Steps>
  <Step title="Open Integrations">
    In Dixa, go to **Settings**, scroll to the **Manage** section, and open **Integrations**.
  </Step>
  <Step title="Add an API token">
    Open the **API tokens** tab and click **Add API token**. Give it a name and select **Dixa API** as the API version, then click **Save changes**.
  </Step>
  <Step title="Copy the token">
    Copy the generated token and store it securely.
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Dixa**, and paste your API token. Leave the API base URL on its default value unless you were given a specific one.

<Warning>
  Keep your API token secret and only paste it into Minimal AI's Dixa integration.
</Warning>
