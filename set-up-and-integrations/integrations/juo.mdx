Connecting Juo lets your AI agents look up subscription data from Juo.

## What you'll need

- **API token** — a Juo API token.

## Find your Juo API token

In Juo, open your account settings and go to the **API / developers** section to create an API token. If you can't find it, contact Juo support.

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Juo**, and paste your API token.

<Warning>
  Keep your API token secret and only paste it into Minimal AI's Juo integration.
</Warning>
