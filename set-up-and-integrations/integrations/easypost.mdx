---

Connecting EasyPost lets your AI agents look up shipments and tracking information from EasyPost.

## What you'll need

- **API key** — your EasyPost production API key, which starts with `EZAK`.

## Find your EasyPost API key

<Steps>
  <Step title="Open API keys">
    Log in to the EasyPost dashboard and go to **Account Settings \> API Keys**.
  </Step>
  <Step title="Copy your production key">
    Copy your **Production** API key (it starts with `EZAK`).
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> EasyPost**, and paste your API key.

<Warning>
  Keep your API key secret and only paste it into Minimal AI's EasyPost integration.
</Warning>
