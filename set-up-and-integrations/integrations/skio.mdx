---

Connecting Skio lets your AI agents look up subscriptions and subscription orders from Skio.

## What you'll need

- **Skio API token** — an API token from your Skio account.

## Find your Skio API token

In Skio, open **Settings** and go to the **API** section to generate an API token. If you can't find it, contact Skio support.

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Skio**, and paste your Skio API token.

<Warning>
  Keep your API token secret and only paste it into Minimal AI's Skio integration.
</Warning>
