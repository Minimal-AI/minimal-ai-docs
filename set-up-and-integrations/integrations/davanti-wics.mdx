---

Connecting Davanti-WICS lets your AI agents look up fulfillment and stock data from your WICS warehouse management system.

## What you'll need

- **API key** and **API secret** — your WICS API credentials.
- **Base URL** — the base URL of your WICS environment.
- **Webshop ID** (optional) — only needed if your account manages multiple webshops.

## Find your Davanti-WICS credentials

Davanti/WICS issues API credentials and a base URL per environment. Request your API key, secret, and base URL from your Davanti or WICS contact if you don't have them.

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Davanti-WICS**, and fill in your API key, API secret, and base URL. Add a webshop ID only if you manage multiple shops.

<Warning>
  Keep these credentials secret and only paste them into Minimal AI's Davanti-WICS integration.
</Warning>
