title: "Print API"
description: "Connect Print API to Minimal AI"
---

Connecting Print API lets your AI agents look up print-on-demand orders and their status from Print API.

## What you'll need

- **Client ID** and **Client secret** — your Print API OAuth credentials.

<Note>
  Print API uses OAuth. Minimal AI builds the token URL automatically, so you only need to provide your Client ID and Client secret.
</Note>

## Find your Print API credentials

Log in to your Print API account and open the **API / account** settings to find your **Client ID** and **Client secret**. Print API provides separate credentials for the test and live environments.

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Print API**, and fill in your Client ID and Client secret.

<Warning>
  Keep your Client secret secret and only paste it into Minimal AI's Print API integration.
</Warning>
