description: "Connect ParcelWILL to Minimal AI"
---

Connecting ParcelWILL lets your AI agents look up shipment and tracking data from ParcelWILL.

## What you'll need

- **API key** — a ParcelWILL API key.

## Find your ParcelWILL API key

In ParcelWILL, open your account settings and go to the **API / integrations** section to generate an API key. If you can't find it, contact ParcelWILL support.

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> ParcelWILL**, and paste your API key.

<Warning>
  Keep your API key secret and only paste it into Minimal AI's ParcelWILL integration.
</Warning>
