Connecting Prediko lets your AI agents look up inventory and forecasting data from Prediko.

## What you'll need

- **API key** — a Prediko API key.

## Find your Prediko API key

In Prediko, open your account settings and go to the **API / integrations** section to generate an API key. If you can't find it, contact Prediko support.

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Prediko**, and paste your API key.

<Warning>
  Keep your API key secret and only paste it into Minimal AI's Prediko integration.
</Warning>
