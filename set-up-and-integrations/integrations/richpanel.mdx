---

Connecting Richpanel lets your AI agents read and reply to conversations in your Richpanel helpdesk.

## What you'll need

- **API key** — an API key created in your Richpanel settings.

## Create your Richpanel API key

<Info>
  Developer API access is available on Richpanel's **Enterprise** plan. If you don't see API Keys, contact your Richpanel account manager.
</Info>

<Steps>
  <Step title="Open API Keys">
    In Richpanel, go to **Settings \> Integrations \> API Keys**.
  </Step>
  <Step title="Generate a key">
    Generate a new API key (or copy an existing one).
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Richpanel**, and paste your API key.

<Warning>
  Keep your API key secret and only paste it into Minimal AI's Richpanel integration.
</Warning>
