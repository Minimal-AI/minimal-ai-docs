---

Connecting Returnless lets your AI agents look up returns and their status from Returnless.

## What you'll need

- **API token** — a Returnless API token.

## Find your Returnless API token

In Returnless, go to **Settings \> Developers** (API) and create an API token. If you can't find it, contact Returnless support.

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Returnless**, and paste your API token.

<Warning>
  Keep your API token secret and only paste it into Minimal AI's Returnless integration.
</Warning>
