Connecting Mirakl lets your AI agents look up marketplace orders and shop data from your Mirakl platform.

## What you'll need

- **Base URL** — your Mirakl platform URL.
- **API key** — your Mirakl API key.
- **Shop ID** (optional) — only needed if you have multiple shops.

## Find your Mirakl API key

<Steps>
  <Step title="Open your user settings">
    In Mirakl, open **My user settings** (your profile).
  </Step>
  <Step title="Copy your API key">
    Find the **API key** section and copy your key. Your **Base URL** is the address you use to access your Mirakl platform.
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Mirakl**, and fill in your base URL and API key. Add a shop ID only if you manage multiple shops.

<Warning>
  Keep your API key secret and only paste it into Minimal AI's Mirakl integration.
</Warning>
