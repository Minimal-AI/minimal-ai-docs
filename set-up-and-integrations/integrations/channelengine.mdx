Connecting ChannelEngine lets your AI agents look up marketplace orders, returns, and product data across every channel you sell on.

## What you'll need

- **API key** — created in your ChannelEngine account.
- **API URL** — your ChannelEngine tenant URL, for example `https://yourtenant.channelengine.net`.

## Find your ChannelEngine API credentials

<Steps>
  <Step title="Open API keys">
    In ChannelEngine, go to **Settings \> API keys** and create (or copy) an API key.
  </Step>
  <Step title="Note your API URL">
    Your API URL is your ChannelEngine tenant address, in the form `https://yourtenant.channelengine.net`.
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> ChannelEngine**, and fill in your API key and API URL.

<Warning>
  Keep your API key secret and only paste it into Minimal AI's ChannelEngine integration.
</Warning>
