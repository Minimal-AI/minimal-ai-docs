Connecting Picqer lets your AI agents look up orders, products, and stock from your Picqer warehouse.

## What you'll need

- **Subdomain** — the `{subdomain}` in `https://{subdomain}.picqer.com`.
- **API key** — a Picqer API key.

## Find your Picqer API key

<Steps>
  <Step title="Note your subdomain">
    Your subdomain is the part before `.picqer.com` in the address you use to open Picqer.
  </Step>
  <Step title="Create an API key">
    In Picqer, open **Settings** and go to the **API keys** section. Create a new API key (or copy an existing one).
  </Step>
</Steps>

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Picqer**, and fill in your subdomain and API key.

<Warning>
  Keep your API key secret and only paste it into Minimal AI's Picqer integration.
</Warning>
