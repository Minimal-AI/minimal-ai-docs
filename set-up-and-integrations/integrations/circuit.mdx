Connecting Circuit lets your AI agents look up route and delivery information from Circuit.

## What you'll need

- **API token** — a Circuit API token.

## Find your Circuit API token

In Circuit, open **Settings** and go to the **Integrations / API** section to generate an API token. If you can't find it, contact Circuit support.

## Connect in Minimal AI

Go to [Dashboard \> Integrations](https://gominimal.ai/dashboard/integrations), click **Add an integration \> Circuit**, and paste your API token.

<Warning>
  Keep your API token secret and only paste it into Minimal AI's Circuit integration.
</Warning>
